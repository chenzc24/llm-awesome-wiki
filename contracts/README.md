# Contracts

This directory contains reusable machine-readable contracts maintained by the
system repository. They are not active workspace data.

Generated knowledge workspaces may copy or reference these contracts under
their own `contracts/` directory. The workspace-local `schema.md` remains the
human and agent-facing structure guide; these files are for deterministic
validation and tooling.
